# 🔐 CipherVault

**ابزار حرفه‌ای مدیریت رمز عبور و رمزنگاری متن**

یک ابزار خط فرمان (CLI) برای تولید رمز عبور امن، بررسی قدرت رمز، و رمزنگاری/رمزگشایی متن با استاندارد AES-256-GCM و پشتیبانی از کلیپ‌بورد.

---

## 🌟 ویژگی‌ها

- **تولید رمز عبور** – ایجاد رمزهای تصادفی و امن (۸ تا ۶۴ کاراکتر)  
- **بررسی قدرت رمز** – تحلیل انتروپی، تنوع کاراکترها، و سطح امنیت  
- **رمزنگاری متن** – با الگوریتم AES-256-GCM و استخراج کلید PBKDF2  
- **رمزگشایی متن** – بازیابی امن پیام‌های رمزنگاری‌شده  
- **پشتیبانی از کلیپ‌بورد** – کپی رمز، کلید، و متن رمزنگاری‌شده با تأیید کاربر  
- **رابط کاربری تعاملی** – تمیز، پاسخگو، و کاربرپسند  
- **امنیت بالا** – بدون رازهای سخت‌کدشده، استفاده از salt و nonce منحصربه‌فرد

### ۱. تولید رمز عبور

تولید رمزهای تصادفی و امن:

- طول قابل تنظیم (۸ تا ۶۴ کاراکتر)  
- انتخاب نوع کاراکترها (حروف کوچک، بزرگ، اعداد، نمادها)  
- استفاده از ماژول `secrets` برای تصادفی‌سازی امن  
- تضمین تنوع کاراکترها  
- تحلیل قدرت با انتروپی و پیشنهادات  

**مثال:**
```
طول رمز عبور (8-64): 16
شامل حروف کوچک؟ (y/n): y
شامل حروف بزرگ؟ (y/n): y
شامل اعداد؟ (y/n): y
شامل نمادها؟ (y/n): y

[SUCCESS] رمز عبور با موفقیت تولید شد!

 aK9$mP2@xL5#qR8! 

تحلیل قدرت:
  سطح: بسیار قوی
  طول: 16 کاراکتر
  انتروپی: ~104 بیت
  حروف کوچک: بله
  حروف بزرگ: بله
  اعداد: بله
  نمادها: بله

آیا رمز عبور در کلیپ‌بورد کپی شود؟ (y/n): y
[SUCCESS] رمز عبور در کلیپ‌بورد کپی شد!
```

### ۲. بررسی قدرت رمز

تحلیل امنیت رمز عبور:

- تحلیل طول  
- بررسی تنوع کاراکترها  
- محاسبه انتروپی (~بیت)  
- سطح‌بندی: ضعیف / متوسط / قوی / بسیار قوی  
- راهنمایی‌های عملی برای بهبود  

**مثال:**
```
رمز عبور را وارد کنید: mypassword123

تحلیل قدرت رمز عبور

رمز عبور: mypassword123

[Medium] سطح کلی: متوسط

طول: 13 کاراکتر
انتروپی: ~51 بیت

انواع کاراکترها:
  بله حروف کوچک (a-z)
  خیر حروف بزرگ (A-Z)
  بله اعداد (0-9)
  خیر نمادهای خاص (!@#...)

پیشنهادات:
  - اضافه کردن حروف بزرگ (A-Z)
  - اضافه کردن نمادهای خاص (!@#$%^&*...)
  - افزایش طول به 16+ کاراکتر برای امنیت بهتر
```

### ۳. رمزنگاری متن

رمزنگاری متن حساس با AES-256-GCM:

- انتخاب کلید تصادفی یا دلخواه (حداقل ۸ کاراکتر)  
- استخراج کلید امن از طریق PBKDF2-HMAC-SHA256 (480,000 تکرار)  
- رمزنگاری احراز هویت‌شده (حالت GCM جلوگیری از دستکاری)  
- خروجی کدشده با Base64  
- کپی کلید و متن رمزنگاری‌شده در کلیپ‌بورد با تأیید کاربر  

**مثال:**
```
متن برای رمزنگاری را وارد کنید: Hello, World!

[SUCCESS] رمزنگاری با موفقیت انجام شد!

متن اصلی:
Hello, World!

کلید رمزنگاری:
 MySecureKey12345 
این کلید را ذخیره کنید! برای رمزگشایی به آن نیاز دارید!

متن رمزنگاری‌شده:
gAAAAABl1x2Y3R...K9m3pQ==

آیا کلید در کلیپ‌بورد کپی شود؟ (y/n): y
[SUCCESS] کلید در کلیپ‌بورد کپی شد!
آیا متن رمزنگاری‌شده در کلیپ‌بورد کپی شود؟ (y/n): y
[SUCCESS] متن رمزنگاری‌شده در کلیپ‌بورد کپی شد!
```

### ۴. رمزگشایی متن

رمزگشایی پیام‌های رمزنگاری‌شده:

**مثال:**
```
متن رمزنگاری‌شده را وارد کنید: gAAAAABl1x2Y3R...K9m3pQ==
کلید رمزگشایی را وارد کنید: MySecureKey12345

[SUCCESS] رمزگشایی با موفقیت انجام شد!

متن اصلی:
 Hello, World! 

آیا متن رمزگشایی‌شده در کلیپ‌بورد کپی شود؟ (y/n): y
[SUCCESS] متن رمزگشایی‌شده در کلیپ‌بورد کپی شد!
```

## 🛠 جزئیات فنی

### رمزنگاری

- **الگوریتم**: AES-256-GCM (حالت Galois/Counter)  
- **استخراج کلید**: PBKDF2-HMAC-SHA256  
- **تعداد تکرار**: 480,000 (توصیه OWASP 2023)  
- **Salt**: 16 بایت، تصادفی برای هر رمزنگاری  
- **Nonce**: 12 بایت، تصادفی برای هر رمزنگاری  
- **خروجی**: کدشده با Base64 (salt + nonce + ciphertext)  

### تولید رمز عبور

- **منبع تصادفی**: ماژول `secrets` (CSPRNG)  
- **مجموعه کاراکترها**: تا 94 کاراکتر قابل چاپ ASCII  
- **انتروپی**: ~6.5 بیت در هر کاراکتر (در صورت استفاده از تمام مجموعه)  
- **تضمین تنوع**: حداقل یک کاراکتر از هر نوع انتخاب‌شده  

### پشتیبانی از کلیپ‌بورد

- **کتابخانه**: `pyperclip`  
- **چندسکویی**: ویندوز، لینوکس، مک  
- **کنترل کاربر**: تأیید صریح قبل از کپی  
- **مدیریت خطا**: بازگشت امن در صورت عدم دسترسی به کلیپ‌بورد  

### ویژگی‌های امنیتی

- تولید اعداد تصادفی امن  
- رمزنگاری احراز هویت‌شده (جلوگیری از دستکاری و حملات تکرار)  
- استخراج کلید امن با تعداد تکرار بالا  
- عدم وجود رازهای سخت‌کدشده  
- عملیات امن حافظه (بدون لاگ‌گیری متن اصلی)  
- تعامل امن با کلیپ‌بورد
